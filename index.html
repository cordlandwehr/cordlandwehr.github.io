Selected Contributions to KDE Projects

Rocs

Rocs is a graph theory simulator that combines a visual editor for creating graphs and a scripting engine that allows performing graph algorithms. Using this simulator, one can easily test and develop graph algorithms. It is especially suited to teach algorithm courses but as well in research when experimenting with algorithms in a drafting stage. From the software development point of view, the most interesting aspects are:

  • the visual editor, which consists of a QtQuickView and integrates into the QtWidget based application
  • the scripting engine, which propagates the graph and its elements to a JavaScript engine
  • the graph document backend, which is a non-trivial example of a Qt class hierarchies with complex interactions

(GitHub is a mere project mirror, the main repository is at git.kde.org)

Artikulate

Artikulate is a pronunciation trainer for perfecting your diction skills. It implements a learning process by repeated self-assessment and -adjustment. From the software development point of view, the most interesting aspects are:

  • it's a hybrid C++/QtQuick application, consisting of a QtQuick frontend and a C++ backend
  • for recording and playing, the sound backend is GStreamer (usued via QtGStreamer)

(GitHub is a mere project mirror, the main repository is at git.kde.org)

KDE on Android SDK

Docker image that provides a self-contained cross-building environment for building Qt/KF5 applications for Android.

Selected Contributions to Research Projects

Robot-Swarm-Simulator

A versatile and scalable simulator for 3D particle swarms that consist of autonomous agents, each individually performing local operations. The simulator was used for groundwork for the following two publications:

  • "Collisionless gathering of robots with an extent", SOFSEM 2011
  • "A new approach for analyzing convergence algorithms for mobile robots", ICALP 2011
